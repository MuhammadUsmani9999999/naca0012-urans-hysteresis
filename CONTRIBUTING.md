# Contributing

Thank you for your interest in contributing to this project. This repository documents a computational fluid dynamics portfolio study; contributions that improve the automation scripts, extend the analysis, or correct errors are welcome.

## How to Contribute

1. **Fork** this repository and create a feature branch from `main`.
2. Make your changes, ensuring all Python code follows PEP 8 conventions.
3. Test any script modifications against a compatible ANSYS Fluent installation.
4. Submit a **pull request** with a clear description of your changes.

## Reporting Issues

If you spot a technical error in the methodology, results, or documentation, please open an issue with:

- A clear title describing the problem.
- The relevant section or file.
- Your suggested correction (if applicable), with references where appropriate.

## Code Style

- Python scripts should be compatible with Python 3.10+.
- Use British English in all documentation and comments.
- Follow the existing naming conventions for consistency.

## Scope

This is primarily a documentation and automation repository. The ANSYS Fluent mesh and case files are not included due to their size; contributors will need access to Fluent 2025 R2 to reproduce results.
